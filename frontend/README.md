# Bitcoin Mining Concentration Map - Frontend

This is the Next.js frontend for the **Bitcoin Mining Concentration Map** application. It provides a real-time analytics terminal and interactive geospatial dashboard built with TypeScript, Leaflet, and Recharts, styled strictly following the Real Rails design guidelines.

## Getting Started

### 1. Configure the Environment
Ensure you have configured your environment variables. You can copy the example file:
```bash
cp .env.example .env.local
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the Development Server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
